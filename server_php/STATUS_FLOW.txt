═══════════════════════════════════════════════════════════════
  ADMIN PANEL TABS (3 TABS ONLY)
═══════════════════════════════════════════════════════════════

1. 📋 SUBMITTED (Needs Review)
   • New KYC submissions
   • Awaiting admin review
   • Actions: Approve ✓ | Reject ✗

2. ✅ VERIFIED
   • Approved KYC records
   • Vendors can work
   • Actions: Revoke ⚠ (if needed)

3. ❌ REJECTED
   • Rejected KYC records
   • Vendors must resubmit
   • Actions: None (vendor resubmits)

═══════════════════════════════════════════════════════════════
  STATUS MEANINGS
═══════════════════════════════════════════════════════════════

SUBMITTED  = Vendor submitted KYC, waiting for admin review
VERIFIED   = Admin approved, vendor can add vehicles and work
REJECTED   = Admin rejected, vendor must fix and resubmit
