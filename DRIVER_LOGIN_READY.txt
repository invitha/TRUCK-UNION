## 🔐 HOW DRIVER LOGIN WORKS:

1. Vendor creates vehicle and sets:
   - Driver Username
   - Driver Password

2. Driver can login using these credentials:
   - Username: (set by vendor)
   - Password: (set by vendor)

3. If driver forgets password:
   - Vendor can edit vehicle
   - Update driver password
   - Share new password with driver
