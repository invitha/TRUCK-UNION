📋 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ Orders not showing?
   → Check vehicle_id is correct
   → Verify order status is "Assigned" not "Delivered"
   → Check API URL in api_service.dart

❌ Status update fails?
   → Verify AL number exists
   → Check vehicle_id matches
   → Verify database credentials in db_config.php

❌ AL numbers not generated?
   → Run SQL migration file
   → Check if column exists: DESCRIBE customer_orders;
