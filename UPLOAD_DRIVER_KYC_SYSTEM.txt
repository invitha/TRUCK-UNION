📋 HOW IT WORKS
═══════════════════════════════════════════════════════════════

1. DRIVER: Opens KYC screen in app
   → Fills personal information
   → Uploads Aadhar (front + back)
   → Uploads PAN card
   → Uploads License (front + back)
   → Fills address
   → Submits KYC

2. SYSTEM: Saves data and documents
   → Status: "submitted"
   → Admin gets notification

3. ADMIN: Reviews KYC in admin panel
   → Views all documents
   → Verifies information
   → Approves or Rejects

4. DRIVER: Gets status update
   → If verified: Can start working
   → If rejected: Can resubmit
