📋 HOW IT WORKS
═══════════════════════════════════════════════════════════════

1. Driver submits KYC
   → Notification created: "KYC Submitted"

2. Admin reviews KYC
   → Changes status to "under_review"
   → Notification created: "KYC Under Review"

3. Admin approves KYC
   → Changes status to "verified"
   → Notification created: "KYC Verified ✓"

4. Driver opens app
   → Sees notification badge
   → Taps to view notifications
   → Reads KYC status update
