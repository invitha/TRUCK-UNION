The dashboard.php tried to SELECT vehicle_tonnage column from the
vehicles table, but that column didn't exist in your database.

This caused the entire page to crash with:
"Fatal error: Unknown column 'vehicle_tonnage'"

═══════════════════════════════════════════════════════════════
  ✅ WHAT I FIXED
═══════════════════════════════════════════════════════════════

1. AUTO-CREATE COLUMN
   - Dashboard now checks if vehicle_tonnage exists
   - If missing, it automatically creates it
   - Column added: vehicle_tonnage VARCHAR(50)

2. CONDITIONAL QUERY
   - Dashboard checks which columns exist
   - Builds SQL query based on available columns
   - Works with or without vehicle_tonnage

3. SAFE ACCESS
   - Only accesses vehicle_tonnage if column exists
   - No crash if column is missing
   - Graceful fallback

═══════════════════════════════════════════════════════════════
  📋 CHANGES MADE TO DASHBOARD.PHP
═══════════════════════════════════════════════════════════════

Change 1: Auto-create column (Line 13-18)
```php
// Check and add vehicle_tonnage column if missing
$tonnage_col_check = $con->query("SHOW COLUMNS FROM vehicles LIKE 'vehicle_tonnage'");
if ($tonnage_col_check && $tonnage_col_check->num_rows == 0) {
    $con->query("ALTER TABLE vehicles ADD COLUMN vehicle_tonnage VARCHAR(50) AFTER vehicle_size_feet");
}
```

Change 2: Check if column exists (Line 217-219)
```php
// Check if tonnage column exists
$tonnage_check = $con->query("SHOW COLUMNS FROM vehicles LIKE 'vehicle_tonnage'");
$has_tonnage_column = $tonnage_check && $tonnage_check->num_rows > 0;
```

Change 3: Conditional SELECT query (Line 222-247)
```php
// Build query based on available columns
if ($has_location_columns && $has_tonnage_column) {
    // Include both location and tonnage columns
} elseif ($has_location_columns && !$has_tonnage_column) {
    // Include only location columns
} elseif (!$has_location_columns && $has_tonnage_column) {
    // Include only tonnage column
} else {
    // Include neither
}
```

Change 4: Safe access to tonnage (Line 1248-1259)
```php
// Count fleet by tonnage (only if column exists)
if ($has_tonnage_column) {
    $tonnage = $v['vehicle_tonnage'] ?: 'Unknown';
    // ... process tonnage data
}
```
