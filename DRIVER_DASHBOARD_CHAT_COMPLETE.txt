HOW IT WORKS:
-------------

1. Driver logs in → Goes to DriverDashboard
2. DriverDashboard shows bottom navigation with 3 tabs
3. Tab 0 (Dashboard): Shows card-based home with 4 action cards
4. Tab 1 (Orders): Shows complete order management (AL numbers, POD, etc.)
5. Tab 2 (Support): Shows chat bot for help and support

CHAT BOT FLOW:
--------------
1. User opens Support tab
2. Bot greets with welcome message
3. Quick reply buttons shown for common topics
4. User taps button or types message
5. Bot responds with relevant information
6. User can continue conversation or select new topic
