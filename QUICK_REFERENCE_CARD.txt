┌──────────────────────────────────────────────────────────────────┐
│                      HOW TO ASSIGN A VEHICLE                      │
└──────────────────────────────────────────────────────────────────┘

1. Open dashboard.php
2. Find vehicle in list
3. Click "📋 Assign" button
4. Fill form in modal
5. Click "✅ Assign Vehicle"
6. Done! ✓


┌──────────────────────────────────────────────────────────────────┐
│                      PAYMENT STATUS QUICK GUIDE                   │
└──────────────────────────────────────────────────────────────────┘

🔴 UNPAID
   When: No payment received
   Advance Field: Hidden
   Example: Credit customer, pay on delivery

🟡 ADVANCE PAID
   When: Partial advance given
   Advance Field: Shown
   Example: 30% advance, rest on delivery

🟠 PARTIALLY PAID
   When: Multiple payments received
   Advance Field: Shown
   Example: Paid ₹30k of ₹50k total

🟢 FULLY PAID
   When: Complete payment received
   Advance Field: Hidden
   Example: Full amount paid upfront


┌──────────────────────────────────────────────────────────────────┐
│                      FORM FIELDS CHEAT SHEET                      │
└──────────────────────────────────────────────────────────────────┘

REQUIRED FIELDS (marked with *)
├─ AL Number *          → Shipment ID (e.g., AL123456)
├─ Pickup Location *    → Full pickup address
├─ Delivery Location *  → Full delivery address
├─ Expected Date *      → Target completion date
├─ Payment Status *     → Select from dropdown
└─ Payment Amount *     → Total amount in ₹

OPTIONAL FIELDS
├─ Assigned By          → Your name
├─ Advance Amount       → If advance/partial paid
└─ Notes                → Special instructions


┌──────────────────────────────────────────────────────────────────┐
│                      PAYMENT CALCULATION                          │
└──────────────────────────────────────────────────────────────────┘

Formula: Remaining = Total - Advance

Examples:
├─ Total: ₹50,000 | Advance: ₹15,000 → Remaining: ₹35,000
├─ Total: ₹30,000 | Advance: ₹10,000 → Remaining: ₹20,000
└─ Total: ₹25,000 | Advance: ₹25,000 → Remaining: ₹0

┌──────────────────────────────────────────────────────────────────┐
│                      VALIDATION RULES                             │
└──────────────────────────────────────────────────────────────────┘

AL Number
├─ Required
├─ Only letters and numbers
└─ Example: AL123456, AL789012

Locations
├─ Required
├─ Any text
└─ Be specific with full address

Date
├─ Required
├─ Must be today or future
└─ Cannot select past dates

Payment Amount
├─ Required
├─ Must be number
├─ Must be greater than 0
└─ Can have decimals (e.g., 15000.50)

Advance Amount
├─ Optional (shown for advance/partial)
├─ Must be number
├─ Cannot exceed total amount
└─ Can be 0


┌──────────────────────────────────────────────────────────────────┐
│                      ERROR MESSAGES                               │
└──────────────────────────────────────────────────────────────────┘

"Please fill out this field"
→ Required field is empty

"Please match the requested format"
→ AL Number has invalid characters

"Value must be greater than or equal to 0"
→ Negative number entered

"Vehicle not found"
→ Database error, contact admin

"Error: ..."
→ Database error, check logs
