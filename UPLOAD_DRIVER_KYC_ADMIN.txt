📋 HOW TO USE
═══════════════════════════════════════════════════════════════

1. Open driver_kyc_admin_panel.php in browser
2. See all driver KYC submissions in table
3. Click "View" button to see details
4. Modal opens with:
   - Personal information
   - Document numbers
   - All uploaded images
   - Status update form
5. Select new status (Under Review, Verified, Rejected)
6. Add rejection reason if rejecting
7. Add admin notes
8. Click "Update Status"
9. Driver gets notification automatically
